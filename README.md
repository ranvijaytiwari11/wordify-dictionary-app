# 🌍 WordAtlas

**WordAtlas** is a modern and fully responsive dictionary web application built using **React.js**. It allows users to explore English words with detailed meanings, pronunciation, and real-world visual context — all in a clean and intuitive interface.

---

## ✨ Features

* 🔍 **Smart Word Search**
  Instantly fetch definitions, parts of speech, and usage examples using the Free Dictionary API.

* 🔊 **Audio Pronunciation**
  Listen to accurate word pronunciations with a single click.

* 🖼️ **Visual Context (Images)**
  Integrated with the Pexels API to display relevant images for better understanding.

* 🌙 **Dark Mode / Light Mode**
  Seamless theme switching for better user experience in all environments.

* 📱 **Fully Responsive Design**
  Optimized for desktop, tablet, and mobile devices using Bootstrap.

---

## 🛠️ Tech Stack

* ⚛️ **React.js** (Hooks: useState, useEffect)
* 🌐 **HTML5 & CSS3**
* 🎨 **Bootstrap 5**
* 🔗 **Axios** (API integration)
* ⭐ **FontAwesome** (icons)

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ranvijaytiwari11/wordatlas.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd wordatlas
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the App

```bash
npm start
```

👉 Open in browser: **http://localhost:3000**
